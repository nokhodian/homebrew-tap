# homebrew-tap

Homebrew tap for [nokhodian](https://github.com/nokhodian)'s apps.

## Install MonoClip

```bash
brew tap nokhodian/tap
brew install --cask mono-clip
```

## Apps

| App | Description |
|---|---|
| [mono-clip](https://github.com/nokhodian/mono-clip) | Blazing-fast macOS clipboard manager |
